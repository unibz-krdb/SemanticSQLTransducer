# Semantic SQL Transducer

## Usage

**Requirements**

- Java 21
- Gradle 8.9

**Build**

`gradle build`

**Test**

Testing requires a running [postgres](https://www.postgresql.org/) instance containing an empty database *ssqlt_test*.

`gradle test`
